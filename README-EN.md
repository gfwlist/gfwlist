## 📝 Reporting New URLs

If you would like to report a new URL, please make sure to read our [Wiki Guidelines](https://github.com/gfwlist/gfwlist/wiki/Cautions) first.

### 🔒 Anonymous Submission
To protect your privacy, **do not** send report emails to any personal email addresses of the maintainers. You can submit reports anonymously to the following secure addresses (replace `#` with `@` when sending):

* 📧 `gfwlistreport#proton.me` (Send to [bocchi810](https://github.com/bocchi810))
* 📧 `gfw#dyi.ng` (Send to [cicku](https://github.com/cicku))

> ⚠️ **Note:** GFWList does not provide real-time updates for submitted URLs. To maintain list quality, we usually conduct thorough testing before adding them to the official list.
